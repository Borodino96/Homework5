Команды Git

  • git -v - версия программы
  • git init - создание / отслеживание локального репозитория
  • git add - сохранение/добавление новых файлов к репозиторию
  • git restore --staged origin master URL: - подключение удаленного репозитория к локальному
  • git commit -m "" - оставить комит
  • git push --set-upstream origin master - запушить локальный репозиторий на подключенный удаленный
  • git clone - скачать на локальный репозиторий копию с удаленного
  • git checkout - переключение между ветками
  • git pull - проверка изменений на удаленном репозитории, скачивание по отношению к локальному
  • git checkout -b - создать новую ветку
  • git log - история комитов
